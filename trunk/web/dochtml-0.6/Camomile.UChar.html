Module Camomile.UChar


module UChar: sig  end
Unicode Characters. All 31bit code points are supported.

type t 
exception Out_of_range
val char_of : t -> char
char u returns the Latin-1 representation of u. If u can not be represented by Latin-1, raises Out_of_range
val of_char : char -> t
char c returns the Unicode character of the Latin-1 character c
val code : t -> int
code u returns the Unicode code number of u. If the value can not be represented by a positive integer, raise Out_of_range
val chr : int -> t
code n returns the Unicode character with the code number n. If n exceeded 31-bit value, raises Invalid_arg
val uint_code : t -> int
uint_code u returns the Unicode code number of u. The returned int is unsigned. that is, on the 32-bit platforms, the sign bit is used for storing the 31-th bit of the code number.
val chr_of_uint : int -> t
uint_code n returns the Unicode character of the code number n. The n is interpreted as unsigned. that is, on the 32-bit platforms, the sign bit is treated as the 31-th bit of the code number.
